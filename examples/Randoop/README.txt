To generate  unit tests for the container class using Randoop, you need to do the following steps:
1- Go to the C:\
1- Create workspace folder 
1- Copy Randoop folder to c:\workspace folder
1- Compile the container class (i.e., mvn compile)
2- Go to the directory container\randooptests
3- run the following command which starts from java -classpath ....
 container\randooptests>java -classpath .;C:\workspace\Randoop\container\target\classes\;c:\workspace\Randoop\randoop-all-3.0.8.jar randoop.main.Main gentests --classlist=c:\workspace\Randoop\container\randooptests\myclasses.txt --timelimit=10 
4- Copy the java files, which are generated by Randoop, from the randooptests and paste them to the src\test\java\container
5- Run mvn test and mvn cobertura:cobertura

Note:
If you decide to change the paths or the folder names, change the command-line accordingly.
